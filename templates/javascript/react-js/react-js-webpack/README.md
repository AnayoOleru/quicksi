# <%= projectName %>

## Quick Start

``` bash
# Install dependencies for server
npm install

# npm start
npm run dev
```

### To test
To run unit tests:

```
npm run test
```

### To build the production package
```
npm run build
```

### Eslint
There is a `.eslintrc.js` config for eslint ready with React plugin.

To run linting, run:

```
npm run lint
```

Open and check package.json scripts to get more info about this
